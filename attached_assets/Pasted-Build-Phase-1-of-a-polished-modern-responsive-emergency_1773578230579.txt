Build Phase 1 of a polished, modern, responsive emergency alert system called "Khurais Emergency Alert System" using Next.js 14 (App Router) + TypeScript + Tailwind CSS + shadcn/ui + lucide-react.

IMPORTANT:
This is Phase 1 only.
Focus on building the core frontend foundation with premium UI quality, clean architecture, reusable components, and realistic mock data.
Do NOT build backend logic yet.
Do NOT build full functionality yet.
Do NOT over-engineer.
Make it look like a real enterprise emergency operations platform demo.

DESIGN STYLE:
- Dark industrial enterprise theme
- Primary red: #EF4444
- Background: dark slate / charcoal
- Cards: slightly lighter gray
- Text: white / cool gray
- State colors:
  - Red = emergency / critical
  - Green = confirmed / safe
  - Yellow = pending / missing
  - Gray = no reply / inactive
  - Blue = info / drill
- Serious, clean, operational look
- Premium SaaS quality
- Strong spacing, hover states, shadows, transitions
- No childish UI
- No generic dashboard template feel

APP CONCEPT:
This system is for a remote industrial site.
A Super Admin sends emergency alerts only to users physically inside selected geofenced zones such as CPF and Camp.
Users receive alerts and confirm their status.
The system tracks:
- Total users
- Confirmed
- Missing
- No reply
